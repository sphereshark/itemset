Item カテゴリ
ItemStack : 手に持ってるアイテムが64個になります。
ItemPhysics : ワールド内に落ちてるアイテムがぺったんこになります。
ModMenu カテゴリ
CustomMenu : 色んなアイテムが設定されてるモッドメニューです。
SpeedBoots : スピードブーツが10%～100%まで設定されてるモッドメニューです。
AddMenu カテゴリ
BlockItem : 建築ページに普段出せないようなアイテムが追加されます。
OrangeArmor : 武器ページにオレンジ色の壊れない防具が追加されます。
Model カテゴリ
Wings : 自分視点でアニメーションの羽が追加されます。
Dev カテゴリ
Exit : ホームメニューへ戻ります。
